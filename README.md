# social
Social network site built with django 3

## Features
1. Register
2. Change password
3. Update profile
4. Login
5. Bookmark external images from other sites
6. Follow and Unfollow other users [Instagram style]
7. Like images
8. Track images ranking
9. Track user actions

## Tools Used
i. Django
ii. Redis
iii. SQLite
iv. PyCharm
v. etc
